# 🛫 Airport Management System

Управление аэропортом с возможностью работы с рейсами, пассажирами, самолетами и билетами через графический интерфейс.

## ✨ Особенности

- 🖥️ **JavaFX GUI** - современный графический интерфейс
- 🗄️ **Firebird Database** - реляционная база данных
- 📊 **CRUD операции** - полное управление данными
- 🏷️ **Вкладки интерфейса** для каждой сущности:
  - 🛩️ Аэропорты
  - ✈️ Самолеты
  - 🧑‍✈️ Пассажиры
  - 🎫 Рейсы
  - 🎟️ Билеты

## 🛠 Технологии

- Java 21
- Firebird Database
- Maven
- JavaFX
- JDBC

## 📋 Требования

- JDK 21+
- Firebird 3.0+
- Maven 3.6+

## 🚀 Запуск проекта

1. **Клонировать репозиторий**
   ```bash
   git clone https://github.com/DKAVrZoV65F/airport.git
   cd Airport
   ```

2. **Настроить базу данных**
   - Установите и запустите Firebird
   - Создайте новую базу данных
   - Обновите параметры подключения в `DbUtil.java`

3. **Запустить приложение**
   ```bash
   mvn clean compile javafx:run
   ```

## 🗂️ Модель данных

База данных включает 5 основных таблиц:
1. `AIRPORT` - аэропорты (IATA коды, местоположение)
2. `AIRCRAFT` - самолеты (модели, производители, вместимость)
3. `FLIGHT` - рейсы (номера, время вылета/прилета)
4. `PASSENGER` - пассажиры (личные данные)
5. `TICKET` - билеты (места, цены)
